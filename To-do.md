## App Web de ouvidoria
App web para registrar, e traduzir as reclamações/elogios da população em dados visuais.

## Funcionalidades de Usuário
- [ ] Registrar reclamação/elogio
- [ ] Consultar reclamação/elogio



## Telas
- [ ] Tel escolha Registro ou consulta
- [ ] Tela de registro identificação
- [ ] Tela de registro categoria
- [ ] Tela de registro detalhes
- [ ] Tela de registro Finalizar
- [ ] Tela de pesquisa manifestação
- [ ] Tela de Dashboard de Ouvidor


## Formulário de Ouvidoria
- [ ] Opção de identificação (Anônimo ou Identificado)
- [ ] Selecionar tipo de manifestação (Elogio, Sugestão, Reclamação, Denúncia)
- [ ] Definir assunto ou categoria (Saúde, Educação, Infraestrutura, etc.)
- [ ] Campo para descrição detalhada do relato
- [ ] Informar local da ocorrência
- [ ] Selecionar data e hora do fato
- [ ] Coleta de dados de contato (E-mail/Telefone)
- [ ] Upload de anexos (Fotos ou Documentos)

## Funcionalidades de Ouvidor
- [ ] Exibir lista de manifestações
- [ ] Filtrar manifestações por tipo, assunto, data, etc.
- [ ] Visualizar detalhes de cada manifestação
- [ ] Exportar manifestações para arquivo
