# Diretriz: Design System (Ouvidoria Digital)

Esta diretiva define a identidade visual e os padrões de componentes para o aplicativo da Ouvidoria Digital, com foco especial em usabilidade para terminais físicos (Totens).

## 1. Identidade Visual (Boituva Material Design)

### Paleta de Cores Oficial
- **Azul Céu (Primário)**: Cor principal para ações positivas e navegação.
- **Marrom Cidade**: Tons terrosos para elementos institucionais.
- **Amarelo Aventura**: Para destaques de atenção ou sugestões.
- **Verde Natureza**: Para logs de sucesso e elogios.
- **Grafite**: Para textos principais, bordas e botões de cancelamento.

### Tipografia
- **Fonte Principal**: `Poppins` (conforme especificado no Icon Set).
- **Estilos**:
    - **Título**: "Boituva Ouvidoria Digital" (Bold, alta legibilidade).
    - **Subtítulo**: "Status da Manifestação" (Semibold).
    - **Corpo**: Texto informativo e descritivo.
    - **Destaque**: Azul Céu para informações críticas como Protocolos.

## 2. Padrões de Componentes (Touch-First)

### Botões (Touch-friendly)
- **Estilo Sólido**: Fundo Azul Céu, texto branco, bordas arredondadas.
- **Estilo Outlined**: Borda Azul Céu, texto Azul Céu (Ex: "VER RESUMO").
- **Estilo Cancelar**: Texto e borda Grafite.
- **Campos de Texto**: Bordas finas Grafite com labels internas claras.

### Componentes de Estrutura
- **Material Design Cards**: Fundo branco puro, sombra suave, cantos arredondados (Ex: Card de Manifestação).
- **Progress Steppers**: Indicadores circulares conectados por linhas verticais.
    - **Completado**: Círculo Azul Céu com check branco.
    - **Ativo**: Círculo Azul Céu vazio (stroke mais grosso).
    - **Pendente**: Círculo cinza claro vazio.

## 3. Acessibilidade (LGPD e Inclusão)
- **Contraste**: Seguir padrões WCAG AA.
- **Avisos de Privacidade**: Checkbox de consentimento LGPD sempre visível no início do registro.
